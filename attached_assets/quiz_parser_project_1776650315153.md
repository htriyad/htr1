# HTML Quiz Parser & Interactive Quiz App

## Description
This project allows you to upload HTML files containing MCQ questions, parse them, and run an interactive quiz.

---

## Files Included

### htmlParser.ts
(Full parser logic)

### global.d.ts
(MathJax typing support)

### upload.tsx
(File upload UI)

### quiz.tsx
(Quiz rendering and logic)

---

## Usage
1. Upload HTML file
2. Questions get parsed
3. Quiz starts automatically
4. Accuracy is calculated

---

## Features
- HTML parsing
- MathJax support
- Image extraction
- Interactive quiz
